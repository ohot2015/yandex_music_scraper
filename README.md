# 🎵 Yandex Music Scraper

Легкий и эффективный скрипт для экспорта треков из плейлистов **Яндекс Музыки** прямо через консоль браузера.

### 🚀 Основные возможности

  * **Обход виртуализации:** Умный сбор данных при прокрутке страницы.
  * **Чистый экспорт:** Формирование списка в удобном формате `.txt`.
  * **Минимализм:** Не требует установки расширений или Node.js.

-----

## 🛠 Инструкция по использованию

Для того чтобы собрать все треки из вашего плейлиста, выполните следующие шаги:

1.  **Откройте плейлист:** Перейдите на страницу нужного плейлиста в [Яндекс Музыке](https://music.yandex.ru).
2.  **Запустите консоль:** Нажмите `F12` (или `Ctrl + Shift + I`) и перейдите на вкладку **Console**.
3.  **Вставьте код:** Скопируйте содержимое файла `index.js` из этого репозитория и вставьте его в консоль.
4.  **Активируйте:** Нажмите `Enter`. На экране появится панель управления (или лог начала захвата).
5.  **Прокрутка:** Медленно или быстро пролистайте плейлист до самого конца, чтобы скрипт «увидел» все треки.
6.  **Завершение:** Нажмите кнопку **Finalize capture** (Завершить захват), которая появится на странице.
7.  **Сохранение:** Браузер предложит скачать готовый `.txt` файл.

-----

## 📄 Формат данных

На выходе вы получите файл со следующим содержанием:

```text
Исполнитель — Название трека
Исполнитель — Название трека
...
```

-----

## ⚠️ Важное примечание

> **Почему нужно листать?** \> Сайт Яндекс Музыки использует **виртуализацию списков**. Это значит, что браузер «видит» только те треки, которые отображаются на экране в данный момент. Скрипт запоминает их по мере появления, поэтому полная прокрутка обязательна для 100% сбора данных.
